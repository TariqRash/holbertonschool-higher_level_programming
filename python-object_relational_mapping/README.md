# Python - Object-relational mapping

This project links Python with MySQL databases using MySQLdb and SQLAlchemy ORM.

## Requirements
- Python 3.8.5
- MySQLdb 2.0.x
- SQLAlchemy 1.4.x

## Files
| File | Description |
|------|-------------|
| 0-select_states.py | Lists all states |
| 1-filter_states.py | States starting with N |
| 2-my_filter_states.py | Filter by user input |
| 3-my_safe_filter_states.py | SQL injection safe |
| 4-cities_by_state.py | All cities with states |
| 5-filter_cities.py | Cities of a state |
| model_state.py | State class definition |
| 7-model_state_fetch_all.py | All states via ORM |
| 8-model_state_fetch_first.py | First state |
| 9-model_state_filter_a.py | States containing 'a' |
| 10-model_state_my_get.py | Get state by name |
| 11-model_state_insert.py | Add Louisiana |
| 12-model_state_update_id_2.py | Update state id 2 |
| 13-model_state_delete_a.py | Delete states with 'a' |
| model_city.py | City class definition |
| 14-model_city_fetch_by_state.py | Cities by state |
