# SQL - Introduction

## Project Overview

This project introduces SQL (Structured Query Language) and MySQL database management. Through hands-on practice, we learn database creation, table manipulation, data queries, and basic SQL operations.

## Learning Objectives

- Understand databases and relational databases
- Master SQL fundamentals and MySQL
- Create and manage databases
- Work with DDL (Data Definition Language)
- Use DML (Data Manipulation Language)
- Execute SELECT, INSERT, UPDATE, DELETE operations
- Implement subqueries and MySQL functions

## Requirements

- MySQL 8.0 on Ubuntu 22.04 LTS
- All SQL keywords in UPPERCASE
- Comments before each query
- File description at the top
