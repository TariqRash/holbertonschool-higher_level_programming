# Python - Everything is object

This project explores how Python handles objects, mutable vs immutable types, references, aliasing, and how variables are passed to functions.

## Learning Objectives

- What is an object
- Difference between a class and an object or instance
- Difference between immutable and mutable objects
- What is a reference, assignment, and alias
- How to know if two variables are identical or linked to the same object
- How to display the variable identifier (memory address)
- Built-in mutable and immutable types
- How Python passes variables to functions

## Requirements

- Python 3.8.5 on Ubuntu 20.04 LTS
- pycodestyle 2.7.*
- Answer files: one line, no shebang, ending with newline
