# Python - Classes and Objects

## Description
This project explores Object-Oriented Programing (OOP) in Python. We learn about classes, objects, attributes, methods, and encapsulation through building a Square class from scratch.

## Learning Objectives
- Understand what OOP is and why it matters
- Know the difference between a class and an object/instance
- Learn about public, protected, and private attributes
- Master the use of properties and getters/setters the Pythonic way
- Understand self and the __init__ method
- Learn about data abstraction and encapsulation

## Requirements
- Python 3.8.5
- pycodestyle 2.7.*
- All files must be executable
- All modules, classes, and methods must have documentation
- No module imports allowed (unless specified)

## Files

| File | Description |
|------|-------------|
| 0-square.py | Empty Square class |
| 1-square.py | Square with private size attribute |
| 2-square.py | Square with size validation |
| 3-square.py | Square with area calculation |
| 4-square.py | Square with property accessors |
| 5-square.py | Square with printing capability |
| 6-square.py | Square with position coordinates |
