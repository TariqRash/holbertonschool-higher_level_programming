# SQL - More queries

Project for learning advanced SQL queries including user privileges, JOIN operations, and database relationships.

## Learning Objectives
- How to create a new MySQL user
- How to manage privileges for a user
- What's a PRIMARY KEY and FOREIGN KEY
- How to use NOT NULL and UNIQUE constraints
- How to retrieve data from multiple tables
- What are subqueries and JOIN operations
